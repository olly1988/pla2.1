Para agregar una imagen a la página, debe usar la etiqueta <img>, que es un elemento vacío (sin etiqueta de cierre). Debe tener el atributo src para indicar dónde encontrar la imagen.


Sono consigliati anche gli attributi alt que da una descripción de la imagen si la imagen está rota, y el atributo de title para dar más información al pasar el cursor sobre la imagen


dependiendo de dónde lo inserte en un párrafo, se mostrará de manera diferente en este caso lo anidamos dentro de un párrafo en medio de un texto


también tiene un atributo de alineación que se usaba comúnmente para indicar cómo deberían fluctuar otros elementos en la página con respecto a la imagen. Esta opción ha sido eliminada. Deben usarse los controles de alineación cssLos valores de alineación pueden ser right, left, bottm, top y middle. El texto se alinea debajo de la foto si el valor asinado es bottom y así sucesivamente con los otros valores
