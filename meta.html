Meta

The tag <meta> is an empty element that have to be written
inside the <head>. It will be not displayed to the users.
It's tell to search engines about the page.
The attributes that are most used with it are the NAME and CONTENT
attributes and usually works togheter. The value of the name can be anything
we want and the most commonly used are description and keywords
that are being used by search engines to understand what the page about.
Instead the value of robots for the name is used to tell whether
to add it to search engines.

Another attribute that is used for the meta tag is http-equiv and as a NAME
attribute it is used in conjunction with the content attribute. With such
attribute usually it's common to use the value author to define the Author.
the values pragma to prevent browser from caching the page
